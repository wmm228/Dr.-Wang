# 欢迎使用 Markdown 编写博客

这是一篇示例博客，用于展示如何使用 Markdown 格式编写内容，并在前端完美展示。

## 1. 文本格式支持

你可以使用 **加粗**，*斜体*，或者 ~~删除线~~。
也可以引用一段话：

> “星光不问赶路人，时光不负有心人。”

## 2. 代码高亮支持

如果你是程序员，你肯定需要代码块：

```python
def hello_world():
    print("Hello, World!")
    return True
```

或者 JavaScript:

```javascript
const greeting = "Hello, World!";
console.log(greeting);
```

## 3. 数学公式支持 (MathJax)

我们需要支持复杂的数学公式，例如：

行内公式：$E = mc^2$

独立公式：

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
