# 🔬 科研经历 (Research Experience)

## 🔭 研究方向 (Research Interests)

我的主要研究兴趣集中在 **AI for Education** 与 **大语言模型 (LLM)** 的交叉领域，致力于通过人工智能技术推动教育的个性化与认知深度发展。

*   🧠 **智能教育 (AI for Education)**: 试题生成与评估、认知深度量化、个性化学习路径推荐。
*   🔗 **多跳问答 (Multi-Hop QA)**: 零样本推理框架、检索增强生成 (RAG)、逻辑推理路径优化。
*   💡 **认知智能 (Cognitive Intelligence)**: 基于 Bloom 分类法的认知层次建模、元认知推理智能体。

## 🏫 科研项目 (Research Projects)

### 🚀 教学相长：CT3.0范式下的检索增强教学案例生成系统
*   🏅 **来源**：第七届中国研究生人工智能创新大赛（全国二等奖）
*   👤 **角色**：核心负责人（Team Leader）
*   🛠️ **工作内容**：
    *   🔹 构建 **CTKG 教学图谱**，覆盖 20 类图谱、1,985 教学单元及 6,464 关系。
    *   🔹 设计 **提示蒸馏 (Prompt Distillation)** 技术，将 Abstract/Theory/Design 三类提示蒸馏至轻量模型。
    *   🔹 实现 **多路径检索 (Multi-path Retrieval)**，并行 Lexical/Semantic/KG 三通道检索，提升案例要素识别精度。
    *   🔹 引入 **滑动窗口 (Sliding Window)** 策略，动态聚焦关键片段，减少 Token 开销。

### 🚀 面向计算学科教育的试题分类模型与知识提取模型
*   🏅 **来源**：华为杯·第六届中国研究生人工智能创新大赛（全国三等奖）
*   👤 **角色**：主要完成人
*   🛠️ **工作内容**：
    *   🔹 构建 **CompEdu Dataset** (2,031道试题)，覆盖 Bloom 分类法知识维度与认知过程维度。
    *   🔹 微调 BERT 模型，实现知识维度分类 Accuracy 96%，认知分类 Accuracy 82%。
    *   🔹 提出基于知识树的自动少样本学习方法，微调 Qwen2-1.5B-Instruct，F1 提升至 68.84%。

### 🎮 基于蓝心千询构建俄罗斯方块 (AIGC 创新赛)
*   🏅 **来源**：中国高校计算机大赛·AIGC 创新赛（全国三等奖）
*   👤 **角色**：核心开发者
*   🛠️ **工作内容**：
    *   🔹 利用蓝心大模型结合 Prompt Engineering 快速生成核心代码框架。
    *   🔹 实现完整游戏逻辑（移动、旋转、碰撞检测等），构建自动化调试流程，将游戏稳定运行率提升至 90%+。

### 🤖 学思并重：简约美学驱动的元认知推理智能体
